# NameForge (AutoName MVP)

Приложение для автоматизированного управления наименованиями и описаниями товаров в **МойСклад** с использованием локальной базы данных **SQLite** и интерфейса **Streamlit**.

## 🚀 Быстрый старт для ИИ
Если вы новая сессия ИИ, начните с чтения этих файлов:
1.  **`docs/HANDOFF.md`** — полная сводка текущего состояния проекта и следующего шага.
2.  **`PROJECT_CONTEXT.md`** — правила домена, статусы, структура БД и хэширование.
3.  **`DEVELOPMENT_PLAN.md`** — архитектура и история разработки (Phase D завершена).

## 🛠 Технологический стек
- **Frontend:** Streamlit
- **Backend:** Python 3.10+
- **Database:** SQLite (без ORM)
- **API:** MoySklad JSON API (через `src/moysklad_client.py`)
- **Validation:** Pydantic v2

## 📂 Структура проекта
- `src/` — исходный код (бизнес-логика, API клиент, БД репозитории).
- `pages/` — страницы интерфейса Streamlit.
- `scripts/` — утилиты для импорта, настройки атрибутов и аудита.
- `data/` — локальная база данных (по умолчанию `autoname.db`).
- `config/` — конфигурационные файлы (карта атрибутов MoySklad).
- `tests/` — юнит-тесты (pytest).

## 📝 Основные команды
- **Запуск UI:** `python -m streamlit run app.py`
- **Импорт из МС:** `python scripts/import_from_ms.py`
- **Запуск тестов:** `pytest`
- **Проверка линтером:** `ruff check .`
